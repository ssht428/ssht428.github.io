Matlab导入数据文件，m脚本中相对路径写法：
假设整个文件夹如下图所示：

m脚本代码如下： 

clear all;
close all;
% import data from file
x = importdata('data.txt');
txt=x.textdata
clear x;
x = importdata('folderB_A\data.txt');
txt=x.textdata
clear x;
x = importdata('..\data.txt');
txt=x.textdata
clear x;
x = importdata('..\folderA\data.txt');
txt=x.textdata
clear x;

输出结果如下：

    'B'

    'B_A'
txt = 
    'parent'
